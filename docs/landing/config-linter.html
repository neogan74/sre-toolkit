Shift-left Checks • YAML • Dockerfiles • Helm

Catch configuration mistakes before they land in the cluster.

config-linter focuses on configuration safety: Kubernetes YAML validation, Dockerfile checks, and the first layer of Helm chart awareness. It is the toolkit’s most build-time-oriented tool and the right place to fail early.
